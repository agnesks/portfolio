CSS Selectors: Target the Right Element

Microlearning Module
Canvas
CodePen
Quizzes
Videos
Articulate Storyline
Context:

This microlearning module is based on a previous HTML+CSS blended learning course. Course outcomes showed that learners struggled to apply CSS selectors in practice. This often led to confusion and frustration when their code didn’t behave as expected. This module focuses on addressing these recurring issues by targeting common beginner mistakes and supporting learners in developing a more practical understanding of selectors through interaction. The module also introduces debugging early, helping learners diagnose why their code doesn’t work.

What I did:

Designed the curriculum, created weekly learning materials, videos and interactive tasks.

Outcome:

High student engagement and positive feedback. Currently redesigning parts of the material as interactive e-learning modules.
